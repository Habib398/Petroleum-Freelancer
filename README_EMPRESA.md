# Checklist rapido para usar en empresa

## 1. Instalacion
- Ejecutar `instalar_empresa.bat`.
- Confirmar que `verificar_proyecto.py` termine en OK.

## 2. Configuracion
- Revisar `.env`.
- Ajustar usuario admin, contrasena, puerto y host si aplica.

## 3. Arranque
- Ejecutar `iniciar_empresa.bat`.
- Confirmar acceso a `/inicio`.

## 4. Validaciones minimas
- Login admin.
- Cambio de contrasena.
- Alta de estaciones y usuarios.
- Revision de permisos por rol.
- Respaldo con `respaldo_manual.bat`.
