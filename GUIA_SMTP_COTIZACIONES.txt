SMTP QUEDA COMO RESPALDO

La opcion recomendada para produccion ahora es AWS SES.

Este archivo se conserva solo como referencia para pruebas locales o respaldo SMTP.

SI QUIERES USAR SMTP COMO RESPALDO
- COG_SMTP_HOST=smtp-mail.outlook.com
- COG_SMTP_PORT=587
- COG_SMTP_USER=usiel54@hotmail.com
- COG_SMTP_PASS=TU_CONTRASENA
- COG_SMTP_FROM=usiel54@hotmail.com
- COG_SMTP_TLS=1

RECOMENDADO
- usa MAIL_PROVIDER=auto o MAIL_PROVIDER=ses
- configura SES_FROM_EMAIL y SES_REGION
- deja SMTP solo como plan B
